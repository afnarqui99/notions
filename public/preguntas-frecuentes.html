Preguntas Frecuentes - NotionLocal

Encuentra respuestas a las preguntas más comunes sobre NotionLocal

¿Qué es NotionLocal?
NotionLocal es un editor local de Notion que te permite crear, organizar y gestionar tus notas, páginas y documentos sin depender de servicios en la nube. Funciona completamente offline y mantiene tus datos privados en tu dispositivo.
¿Necesito una cuenta para usar NotionLocal?
No, NotionLocal no requiere crear una cuenta. La aplicación funciona completamente offline y almacena tus datos localmente en tu navegador o en una carpeta de tu sistema de archivos que tú elijas.
¿Dónde se guardan mis datos?
Por defecto, tus datos se guardan en el almacenamiento del navegador (localStorage/IndexedDB). También puedes configurar la aplicación para guardar tus datos en una carpeta local de tu sistema de archivos. Puedes cambiar esta configuración en cualquier momento desde el menú de Configuración.
¿Puedo usar NotionLocal sin conexión a internet?
Sí, NotionLocal funciona completamente offline. Una vez que cargas la aplicación en tu navegador, puedes usarla sin necesidad de conexión a internet. Todos tus datos se almacenan localmente.
¿Qué funcionalidades incluye NotionLocal?
NotionLocal incluye una amplia gama de funcionalidades:
  • Editor de texto enriquecido con formato completo
  • Tablas dinámicas estilo Notion
  • Tableros Kanban para gestión de proyectos
  • Cliente API (Postman) integrado
  • Editor Visual Code para proyectos completos
  • Terminal Node.js y Python integrada
  • Calendario interactivo de eventos
  • Galerías de imágenes y archivos
  • Editor Markdown con vista previa
  • Conexión a bases de datos (PostgreSQL, MySQL, SQL Server)
¿Cómo creo una nueva página?
Puedes crear una nueva página de varias formas:
  • Haz clic en el botón "+" en el sidebar
  • Usa el botón "Crear tu primera página" en la pantalla de bienvenida
  • Usa el atajo de teclado correspondiente
¿Cómo inserto comandos y funcionalidades?
Para insertar comandos y funcionalidades, haz clic en el botón flotante que aparece cuando escribes en una página. Este botón sigue tu cursor y te permite acceder a todas las funcionalidades disponibles como tablas, Postman, Visual Code, terminales, etc.
¿Puedo exportar mis páginas?
Sí, puedes exportar tus páginas en diferentes formatos:
  • PDF - Para compartir o imprimir
  • Markdown - Para usar en otros editores
  • HTML - Para visualización web
La opción de exportar está disponible en la barra de herramientas de cada página.
¿Mis datos son privados y seguros?
Absolutamente. NotionLocal funciona completamente offline y todos tus datos se almacenan localmente en tu dispositivo. No enviamos ningún dato a servidores externos. Si eliges guardar en una carpeta local, tienes control total sobre dónde se almacenan tus archivos.
¿Puedo usar NotionLocal en múltiples dispositivos?
Si guardas tus datos en una carpeta local y sincronizas esa carpeta entre dispositivos (usando servicios como Dropbox, Google Drive, o sincronización manual), puedes acceder a tus páginas desde diferentes dispositivos. Sin embargo, NotionLocal no incluye sincronización automática en la nube.
¿Hay una versión de escritorio disponible?
Sí, NotionLocal está disponible como aplicación de escritorio usando Electron, lo que te permite usar todas las funcionalidades avanzadas como acceso al sistema de archivos, terminales Node.js y Python, y conexión a bases de datos.
¿Necesito instalar algo para usar NotionLocal?
No necesitas instalar nada si usas la versión web. Simplemente abre la aplicación en tu navegador. Para la versión de escritorio, necesitas descargar e instalar la aplicación Electron.
¿NotionLocal es gratuito?
Sí, NotionLocal es completamente gratuito. La aplicación se mantiene gracias a la publicidad de Google AdSense, que es discreta y no interfiere con tu experiencia de uso.
