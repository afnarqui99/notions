Ejemplo 4: Servicios e Inyección de Dependencias

¿Qué es un Servicio?

Un servicio en Angular es una clase que contiene lógica reutilizable y datos compartidos entre componentes.

Mensaje del servicio: {{ mensaje }}

Usuarios desde el Servicio

  • ID: {{ usuario.id }} - Nombre: {{ usuario.nombre }}

Conceptos Aprendidos:

  • @Injectable: Decorador que marca una clase como servicio
  • Constructor Injection: Inyectar servicios en el constructor
  • ngOnInit: Lifecycle hook que se ejecuta después de la inicialización
  • Servicios: Comparten datos y lógica entre componentes
  • Dependency Injection: Angular inyecta automáticamente las dependencias

Código del Servicio (data.service.ts):

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private usuarios = [
    { id: 1, nombre: 'Juan' },
    { id: 2, nombre: 'Ana' }
  ];

  obtenerUsuarios() {
    return this.usuarios;
  }

  agregarUsuario(usuario: any) {
    this.usuarios.push(usuario);
  }
}
